## Deployment on Termux

To deploy this on termx

```bash
  apt update
  apt upgrade
  git clone https://github.com/CodeAx1avek/clone-website-using-termux.git
  ls
  cd clone-website-using-termux
  ls
  python script.py
```

## Deployment on Kali Linux
To deploy in kali 
```bash
  apt-get update
  apt-get upgrade
  git clone https://github.com/CodeAx1avek/clone-website-using-termux.git
  ls
  cd clone-website-using-termux
  python script.py
```
